- Setup Roku Integration -

Step 1: Add the Roku Integration to your HA Server

Use the button below or follow the HA Roku Instructions

Step 2: Ensure Home Assistant can talk to your Roku

If you recive this error make sure your Roku is on and that you enabled "Control by mobile apps" in your Roku Settings.

- Configure Roku Intergration -

Step 1: Go to the Roku Integration

Select Configure on the device you would
like to setup with Media Assistant.

Step 2: Change the App ID to 782875

By defualt the App ID is set to the discontinued PlayOnRoku App for backwards compability with older Roku's.

In order to use Media Assistant you have to change it to Media Assistants App ID: 782875

For more information

Check out the PlayOnRoku section of the
Home Assistant Roku docs: PlayOnRoku Docs

- Play Some Media :D -

In the Media tab select your device and some media to play.

If everything is setup right your Roku should launch
Media Assistant and start playing your media.

For more information and advanced usage

Check out the Home Assistant Roku docs
Roku docs: HA Roku Docs
