- Setup -

Step 1: Install Media Assistant

You can also install by searching
for "Media Assistant" on your Roku Search.

Roku Channel Store Sideload Instructions

Step 2: Ensure control by mobile apps is enabled

In order for apps to talk/send media to Media Assistant you must ensure mobile app control is enabled.

To check this go to your Roku's settings and navigate to

Settings -> System -> Advanced system settings
-> Control by mobile apps -> Network access

and check to see if Network access is set to Enabled.

Step 3: Enjoy Media Assistant :D

Try out some of the Supported Apps

Or if your a Developer check out the dev docs.
